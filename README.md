# <a name="net-api-reference-docs"></a>Documentación de referencia de API de .NET

Este repositorio contiene documentación sobre la referencia de API de .NET.

El repositorio principal de documentación de .NET es el [repositorio de documentación de .NET](https://github.com/dotnet/docs). Vea la [guía de contribución](https://github.com/dotnet/docs/blob/master/CONTRIBUTING.md) para obtener información sobre cómo actualizar la documentación de la referencia de API de .NET.
El seguimiento de todo el trabajo para este repositorio se realiza mediante [problemas de GitHub](https://github.com/dotnet/dotnet-api-docs/issues). La documentación de las API se genera a partir del texto de este repositorio y los ejemplos del repositorio [dotnet/samples](https://github.com/dotnet/samples). Puede seleccionar la etiqueta [Repositorio: ejemplos](https://github.com/dotnet/docs/issues?q=is%3Aopen+is%3Aissue+label%3A%22%3Afile_folder%3A+Repo+-+samples%22) para ver los problemas relativos al código de ejemplo.

Este proyecto ha adoptado el Código de conducta convenido para contribuyentes con el fin de establecer el comportamiento que se espera en nuestra comunidad. Para obtener más información, consulte el [Código de conducta de .NET Foundation](https://dotnetfoundation.org/code-of-conduct).
